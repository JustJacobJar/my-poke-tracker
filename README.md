# My Poke Team Creator

My Poke Team Creator is a place where users can view and create their own poke teams and describe how to use them.

## Tech Stack

- React + Next.js
- TailwindCSS
- Prisma
- Tanstack Query
- Auth.js
- Neon
- Vercel
